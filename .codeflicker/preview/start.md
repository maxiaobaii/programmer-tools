# JSON Pro Tools 启动指南

## 项目概述
这是一个基于 Vue 3 + Vite 的前端 JSON 工具站，JSON 转换工作台页面路由为 `/json`。
本次用于验证页面布局问题时，建议直接启动开发环境并访问 JSON 工具页查看底部留白情况。

## . - JSON Pro Tools

### 快速启动

```bash
cd .
npm run dev
```

**启动后访问**：打开开发服务器日志中显示的本地地址，并访问 `/json` 路由进行验证。

```yaml
subProjectPath: .
command: npm run dev
cwd: .
port: null
previewUrl: null
description: Vue 3 + Vite 单页应用，JSON 工具页面路由为 /json，用于验证页面底部空白和实际布局表现
```
