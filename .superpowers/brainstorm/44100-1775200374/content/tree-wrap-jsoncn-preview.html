视觉特征

  • key 列位置固定
  • 冒号和 value 在同一组内
  • value 自己负责换行，不把后续字段整体往右推

  • 只需要改 `TreeNode.vue` primitive 行布局
  • 把当前单行 flex 改成「key 列 + value 区」布局
