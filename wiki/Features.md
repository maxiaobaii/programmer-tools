# Features

## JSON 工作台

当前站点的核心能力集中在 `#/json` 页面。

### 已支持功能

- JSON 格式化
- JSON 压缩
- JSON 语法校验
- JSON Schema 子集校验
- JSON 树视图
- JSON Diff 对比
- JSONPath 查询
- TypeScript 类型生成
- Java 类型生成
- 多格式转换

## 多格式转换

### 输入支持

- JSON
- YAML
- XML
- CSV
- TOML
- URL Query

### 输出支持

- JSON
- YAML
- XML
- CSV
- TOML
- URL Query
- TypeScript
- Java
- Diff 摘要文本

## 文件与剪贴板能力

- 导入本地文件
- 复制输出结果
- 下载结果文件
- 从剪贴板读取文本

## 首页能力

- 工具卡片入口
- 已上线与开发中状态展示
- 后续工具扩展位

## 404 页面

项目在 GitHub Pages 环境下提供自定义 404 页面：

- Vue Router 兜底路由
- `public/404.html` 错误地址桥接
- 一键返回首页
